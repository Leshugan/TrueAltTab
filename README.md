ENG
You’ve probably noticed that minimizing a game in Windows can be a real headache.
Some games refuse to minimize at all, while others pop back up the moment you select another window — like Chrome or File Explorer.
This script-utility fixes that. When you press Alt+Tab or Win+D, games will minimize and stay minimized, unable to regain focus on their own. In a sense, it works like the old Windows 7 Alt+Tab behavior — which is why I call it True Alt+Tab.

Installation
Just run the executable once — that’s it.

Uninstallation
Run the executable again.

RU
Вы наверняка замечали, что свернуть игру в Windows — та ещё задача.
Одни игры вообще не сворачиваются, другие — самопроизвольно разворачиваются, как только вы выбираете другое окно, например Chrome или Проводник.
Эта скрипт-утилита решает проблему: при нажатии Alt+Tab или Win+D игры сворачиваются намертво и больше не могут самопроизвольно вернуть себе фокус. По сути, работает как старый добрый Alt+Tab из Windows 7 — поэтому я и назвал её Настоящий Alt+Tab.

Установка
Просто запустите исполняемый файл один раз.

Удаление
Запустите исполняемый файл повторно.
